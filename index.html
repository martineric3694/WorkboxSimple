Cache Strategies

  • Cache First: Gambar akan di-cache dan diambil dari cache lebih dahulu.
  • Stale While Revalidate: File CSS dan JS akan langsung diambil dari cache, lalu diperbarui.
  • Network First: API JSON akan diambil dari jaringan terlebih dahulu.
  • Network Only: Semua request lain hanya menggunakan jaringan.
